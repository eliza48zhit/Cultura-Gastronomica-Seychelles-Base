🛠️ Especificaciones Técnicas del ADN de Seychelles
Este contrato documenta la termodinámica de la fusión y la cadena de suministro marina:

Nivel de Especias (uint256): Cuantifica la intensidad de la infusión aromática (curry, jengibre, canela). Este parámetro es esencial para la conservación biológica y la complejidad del perfil de sabor criollo.

Frescura Marina (uint256): Mide las horas transcurridas desde la captura. Este dato es crítico en la blockchain para certificar la calidad de las proteínas antes de su procesamiento térmico.

Usa Leche de Coco (bool): Validador de la base lipídica que actúa como agente emulsionante para integrar los sabores volátiles de las especias.

Sistema de Feedback: Funciones de darLike y darDislike para la interacción comunitaria.
